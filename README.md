# 📘 LKM IF Kelompok 15

Website profil modern dan interaktif untuk **Kelompok 15 LKM IF Angkatan 2025** – sebuah website kelompok studi Informatika yang menampilkan anggota tim dan materi pembelajaran.

**🔗 Situs Langsung:** [lkm-kel-15.vercel.app](https://lkm-kel-15.vercel.app/)

## ✨ Fitur

- 🎨 Antarmuka glassmorphism modern dengan animasi halus
- 👥 Profil tim dengan integrasi LinkedIn
- 📱 Responsif di semua perangkat
- 🎬 Animasi kartu saat di-scroll (scroll-triggered)
- 🧭 Navigasi tetap dengan perubahan gaya saat discroll

## 🛠️ Teknologi yang Digunakan

- **HTML5** – Struktur semantik
- **CSS3** – Properti kustom, flexbox, grid, animasi, efek glassmorphism
- **JavaScript (ES6)** – Intersection Observer API untuk animasi scroll, styling dinamis pada header
- **Vercel** – Hosting dan deployment
